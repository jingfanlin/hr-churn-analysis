# HR员工数据分析系统

## 项目简介
基于 Python + SQLite 构建的HR员工数据分析系统，模拟30名员工的企业人力资源数据库，涵盖9个分析维度，输出结构化分析报告。

## 技术栈
- Python 3.12
- SQLite（建表、多表JOIN、聚合查询）
- Pandas（数据处理与格式化输出）

## 数据库结构
共5张数据表：员工基本信息、部门信息、薪资记录、考勤记录、绩效评估

## 分析内容
1. 各部门人员分布与薪资对比
2. 性别与年龄分布
3. 薪资区间结构分析
4. 员工工龄分布（TOP10排名）
5. 离职率与离职原因分析
6. 薪资调整记录与涨幅统计
7. 绩效等级分布与部门对比
8. 高薪员工 TOP10
9. 综合数据看板（关键指标汇总）

## 主要结论
- 产品部平均薪资最高（15700元），绩效优秀率80%
- 技术部离职率20%，为各部门最高
- 晋升加薪平均涨幅22.5%，高于年度调薪的18.48%

## 运行方式
